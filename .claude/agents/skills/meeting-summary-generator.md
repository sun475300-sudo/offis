# Skill: meeting-summary-generator

## 트리거 키워드
`/skill meeting-summary`, "회의록 써줘", "미팅 요약해줘", "녹취 정리해줘", "액션 아이템 뽑아줘"

## 목적
회의 녹취록, 노트, 또는 대화 내용을 구조화된 회의록으로 변환합니다. 핵심 결정사항과 액션 아이템을 명확히 추출합니다.

## 파이프라인
```
analyst(내용 분석 + 화자/주제 분류)
    ↓
작성자(회의록 작성 — 표준 형식)
    ↓
운영자(파일 저장: docs/meetings/YYYY-MM-DD_주제.md)
```

## 실행 방법
```bash
/skill meeting-summary                          # 클립보드 또는 첨부 파일
/skill meeting-summary --file meeting_notes.txt
/skill meeting-summary --lang en                # 영어 출력
/skill meeting-summary --format slack           # Slack 공유용 압축 형식
```

## Analyst 단계: 내용 분류
```
파악 항목:
1. 참석자 목록 (발언자 식별)
2. 주요 토픽별 분류
3. 결정된 사항 vs 논의 중 사항
4. 액션 아이템 (담당자 + 기한 포함)
5. 다음 미팅 일정
6. 미해결 이슈/리스크
```

## 운영자 단계: 저장
```bash
mkdir -p docs/meetings
FILENAME="docs/meetings/$(date +%Y-%m-%d)_$(echo $TOPIC | tr ' ' '_').md"
cat > $FILENAME << 'CONTENT'
[생성된 회의록]
CONTENT
echo "회의록 저장: $FILENAME"
git add $FILENAME
```

## 출력물
- `docs/meetings/YYYY-MM-DD_주제.md`
- 액션 아이템 목록 (담당자·기한 포함)
- Slack 공유용 압축 요약
- 이메일 발송용 버전 (요청 시)
