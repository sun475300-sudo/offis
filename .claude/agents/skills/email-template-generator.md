# Skill: email-template-generator

## 트리거 키워드
`/skill email`, "이메일 써줘", "메일 초안 만들어줘", "비즈니스 이메일", "이메일 템플릿"

## 목적
컨텍스트(수신자·목적·톤)를 분석하여 즉시 발송 가능한 비즈니스 이메일을 생성합니다. 협업 요청, 제안, 팔로업, 보고 등 모든 비즈니스 상황 지원.

## 파이프라인
```
analyst(상황 분석 + 핵심 메시지 정의)
    ↓
writer(이메일 본문 작성 — 3가지 버전: 간결/표준/상세)
    ↓
번역가(필요 시 영어 버전 추가)
```

## 실행 방법
```bash
/skill email --type 협업요청 --to "파트너사 CTO" --subject "AI 파이프라인 공동 개발"
/skill email --type 팔로업 --context "지난주 미팅 후 제안서 발송"
/skill email --type 보고 --to "팀장" --context "이번 주 진행 상황"
/skill email --lang en  # 영어 버전
```

## 이메일 유형별 템플릿

### 협업 요청
```
Subject: [제안] {프로젝트명} 협업 논의 요청 — {보내는 사람 회사}

안녕하세요 {이름}님,

{공통점/접점 1줄}: [어디서 알게 됐는지, 공통 관심사]

저는 {회사/역할}이며, {핵심 가치 제안 1줄}을 진행 중입니다.

{상대방에게 어떤 이점이 있는지 2-3줄}

30분 정도 이야기 나눌 수 있을까요?
[캘린더 링크 또는 가능한 시간대]

감사합니다,
{이름} | {직함} | {연락처}
```

### 팔로업
```
Subject: Re: {이전 주제} — 다음 단계 확인

안녕하세요 {이름}님,

{날짜} 미팅/연락 이후 안부 여쭙습니다.

논의하셨던 {핵심 사항}에 대해:
- {진행 상황 1}
- {진행 상황 2}

{다음 단계 또는 질문}

언제든 편하게 연락 주세요.
```

### 진행 상황 보고
```
Subject: [주간 보고] {프로젝트명} — {날짜}

{이름}님,

이번 주 진행 사항을 공유드립니다.

✅ 완료
- {항목 1}
- {항목 2}

🔄 진행 중
- {항목}: {진척도}%

⚠️ 이슈/리스크
- {이슈}: {대응 방안}

다음 주 계획: {목표}

문의 사항 있으시면 알려주세요.
```

## Analyst 단계: 컨텍스트 분석
```
분석 항목:
1. 수신자 관계 (처음/기존, 상하관계)
2. 목적 (정보 전달/요청/설득/관계 유지)
3. 긴급도 (즉시/이번 주/여유)
4. 원하는 톤 (격식/친근/간결)
5. 기대하는 액션 (회신/미팅/승인/참고)
```

## Writer 단계: 3가지 버전 출력
```
[간결 버전] — 5줄 이내, 핵심만
[표준 버전] — 10-15줄, 완전한 맥락
[상세 버전] — 20줄+, 배경 설명 포함
```

## 출력물
- 이메일 초안 (최대 3가지 버전)
- 제목 라인 3개 옵션
- 발송 타이밍 추천 (요일/시간대)
- 영어 버전 (요청 시)
